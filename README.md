## What is CPU?
- CPU is the brain of the computer.
- Executes instructions from programs.
- Performs basic operations: addition, subtraction, comparison, data movement.

## Key Concepts
- CPU processes data from RAM.
- Without CPU, computer cannot function.
- Works with motherboard, RAM, and storage to run programs.

## Analogies
- CPU is like a chef in a kitchen: takes ingredients (data), follows recipes (instructions), produces dishes (results).

## Summary
- CPU = brain of computer.
- Executes instructions step by step.
- Essential for all computing tasks.
